# ShopSphere Project

Full-stack e-commerce platform

## Quick Start

### Backend
```bash
cd backend
npm install
cp .env.example .env
npm run seed
npm run dev
```

### Frontend
```bash
cd frontend
npx http-server -p 3000
```

## Docker
```bash
docker-compose up -d
```

Access at: http://localhost
